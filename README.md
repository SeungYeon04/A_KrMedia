## 🖥️ 프로젝트 소개

> 년도별 디자이너, 프로젝트 소개
> 파이어베이스 스토리지 사용
>
> 시작페이지에 있는 2022, 2023...  
> 년도를 선택하여 들어가면  
> 인트로 페이지와 함께 카테고리 표시  
> 그리고 상단페이지는 년도에 따라  
> 회차표시 되도록 하기  
>   
> 년도별 학생들 졸업작품 데이터를  
> Json으로 렌더링하여  
> 학생들과 작품들을 표시
